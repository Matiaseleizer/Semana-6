## 📘 Descripción general del sistema
Este es un programa académico desarrollado en **Java** bajo el paradigma de **Programación Orientada a Objetos (POO)**. El sistema simula la gestión y catalogación de diferentes tipos de experiencias turísticas utilizando conceptos clave como herencia, polimorfismo y encapsulamiento.

---

## 🧱 Estructura general del proyecto

El proyecto está organizado en los siguientes paquetes:

* **`ui`**: Contiene la clase principal para la ejecución del sistema.
* **`model`**: Contiene las clases que van conectadas entre si como ServicioTuristico que es la super clase
* **`data`**:
  * `GestorServicios`: contiene toda la informacion para las distintas clases.
